## Find Me a Table

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Used in the project:
TECH
- Axios
- React
- NextJS
- Prisma
- PostgreSQL
- TypeScript
- Tailwind
- Material UI
- react-simple-image-viewer
- validator
- bcrypt
- jose
- jsonwebtoken

SERVICE
- Neon
- Vercel

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
